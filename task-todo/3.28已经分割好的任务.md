## 📋 Task 3: 单题提取 MVP 测试 (Single Question VLM Extraction)
**Status:** [ ] TODO
**Dependency:** Task 2
**Input:** - `ap-calculus-bc-2019-intl.schema-example.json` (作为输出结构的绝对参考)
- `./dataset/raw_pages/` 中的任意一张单选题图片（例如第 3 页）。
**Action:**
1. 编写一个 Python 脚本，调用视觉大模型 API (Gemini 1.5 Pro 或 Claude 3.5 Sonnet)。
2. 传入图片，并注入以下 **System Prompt**：
   > "你是一个资深的 AP 微积分 BC 试卷数字化专家。请阅读图片，严格按照我提供的 JSON Schema 提取题目数据。
   > **规则：**
   > 1. 所有数学变量、公式、积分必须使用纯正 LaTeX 并用 `$` 或 `$$` 包裹（如 $\int x dx$）。
   > 2. 忽略页码、'GO ON TO THE NEXT PAGE' 等排版杂质。
   > 3. 选择题请将 A, B, C, D 选项剥离前缀，填入 `options` 数组。
   > 4. 若有配图，在 `question` 文本中插入 `<img src=\"images/TODO.png\" alt=\"题目配图\" />`。
   > 5. 根据图片顶部判断 `is_calculator_allowed` (Part A=false, Part B=true)。
   > 6. 仅输出合法 JSON，勿带 Markdown 代码块标记。"
**Output:** `test_question_01.json`
**Validation:** 人工或写脚本校验：是否为严格的 JSON？数学公式是否被正确包裹为 LaTeX？
